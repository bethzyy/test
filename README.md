# 职位抓取助手
这是一个简单实用的招聘信息抓取工具，专门用来筛选与软件开发相关的职位。当你面对大量招聘页面，想快速找出真正需要编程能力的岗位时，这个工具能帮你省去不少功夫。

## 它能做什么
- **静态页面抓取**：用 requests + BeautifulSoup 处理普通的招聘页面，速度快，但是动态渲染的网页有时信息抓不全
- **动态页面抓取**：用 Selenium 处理需要 JavaScript 渲染的页面，速度慢，但结果比较有保障
- **智能筛选**：根据职位名称和描述内容自动识别软件开发相关岗位

## 怎么使用
### 第一步：准备
1. 安装需要的 Python 库：
```bash
pip install requests beautifulsoup4 selenium
```

2. 确保你有 Chrome 浏览器（Selenium 需要）

### 第二步：填写网址
把你想检查的招聘网址，一行一个地写在 `joburl.txt` 文件里。

### 第三步：运行抓取
**想抓静态页面就运行：**
```bash
python job_scraper.py
```

**需要抓动态页面就运行：**
```bash
python dynamic_job_scraper.py
```

工具会自动分析每个网址，告诉你哪些是软件开发职位，哪些不是。

### 第四步：查看结果
程序运行完后会显示：
- ✅ 匹配的职位（软件开发相关）
- ❌ 不匹配的职位
- 每个职位的匹配理由

## 关键词可以自己改
所有的筛选关键词都放在 `config.py` 文件里。如果你觉得有些词应该被识别但没被识别，或者想添加新的技术方向（比如加上“数据分析”、“测试工程师”等），直接修改这个文件里的 `SOFTWARE_JOB_KEYWORDS` 列表就行。

## 一点小提示
1. 有些招聘网站有反爬虫机制，建议合理设置请求间隔
2. Selenium 版本需要 Chrome 浏览器支持，确保 Chrome 已安装
3. 这只是个辅助工具，筛选结果仅供参考，具体职位还是要自己点进去看看

## 文件说明
- `job_scraper.py` - 基础爬虫，速度快，适合静态页面
- `dynamic_job_scraper.py` - Selenium爬虫，能处理动态页面
- `config.py` - 配置文件，可以自定义关键词
- `joburl.txt` - 存放要检查的招聘网址
